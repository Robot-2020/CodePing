<p align="center"><strong>Ping 在，你就不用盯着。</strong></p>

CodePing 是一个面向 AI 编程场景的桌面陪伴应用。它常驻桌面，实时感知 agent 的工作状态，让你在长任务执行时不必一直盯着终端。

## 它能做什么

- 展示思考、打字、执行、等待、完成等实时状态
- 在桌面上提示权限请求和关键进度变化
- 支持主题切换与轻量托盘控制
- 帮你在不守着会话的情况下及时注意重要结果

## 快速开始

```bash
npm install
npm start
```

## 技术栈

- Electron
- Node.js
- 面向 AI coding agent 的本地 hooks
